# rails-carrier-wave

Rails app, generate scaffold Image and adds CarrierWave Uploader
## environment

```shell script
ruby: 2.7.0
rails: 6.0.2.1
mysql: 5.7.12
```

## getting started

```shell script
$ docker-compose up --build
$ docker-compose exec app rails db:create
$ docker-compose exec app rails db:migrate

# access localhost:3000
```
